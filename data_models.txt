Data Models for KidReads

User
  user_type_id
  username
  email
  first_name
  last_name (only initial if Kid)
  city
  state
  country
  birthday
  password_digest
  points (based on # pages read * readingLevel / gradeLevel)

UserType (kid, parent, teacher, admin)
  label
  description

Club
  leader_id
  club_name
  club_type (classroom, family)

ClubMembership
  member_id
  club_id
  membership_type (leader, kid, adult)

Comment
  club_id
  commenter_id
  parent_comment_id
  body

Review
  book_id
  user_id
  rating
  review

BookStatus
  user_id
  book_id
  status: (wish, current, read)

Book
  --> accessed through GoodReads API
  isbn
  title
  author
  pub_year
  genre
  num_pages
  --> might have to be scraped from Scholastic website
  reading_level
